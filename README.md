# 自定义View
#### 尝试着造一些简单的轮子，理解一下如何自定义view
## 1.超简版CircleImage
- 就是实现了圆形头像基本功能，外加了头像外框
- 需要复制attrs那个xml文件
- 两个自定义属性，外框的宽度KuangWidth和外框的颜色KuangWidth(但是如果外框宽度设置太大，会出现bug)
- 可以padding内边距
- 了解了canvas,paint基本用法，简单的自定义view
- Drawable里面bitmapDrawable和bitmap的转换,bitmapshader用法
